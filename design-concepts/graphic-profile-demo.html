Designprinciper

1. Minimalism först
Varje element måste ha ett syfte. Ta bort allt onödigt. Fokus på innehåll och funktion.
2. Endast gråtoner
Inga färger används. Olika nyanser av grått skapar hierarki och visuell struktur.
3. Generöst med whitespace
Låt element andas. Använd marginal och padding för att skapa lugn och överskådlighet.
4. Subtila animationer
Animationer ska förstärka användarupplevelsen, inte distrahera. Smooth transitions och fade-effekter.
5. Typografi som hierarki
Använd font size, weight och letter spacing för att skapa tydlig visuell hierarki.
6. Konsekvent spacing
Använd ett spacing-system (8px, 16px, 24px, 32px, 40px) för harmonisk layout.
