# Changelog

All notable changes to the CivicAI project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Complete project structure reorganization
- Comprehensive documentation system in `/docs`
- Firebase configuration and setup guides
- Automated setup, build, and lint scripts
- Schema validation helper for Firestore collections
- API documentation with complete endpoint reference
- Data model schemas for all Firestore collections
- Firebase Admin SDK configuration
- Firebase Web SDK configuration
- Contributing guidelines
- Footer link to API documentation

### Changed
- Moved all documentation to `/docs` with organized subdirectories
- Moved all HTML demos to `/demos`
- Moved prototypes and design concepts to `/demos`
- Completely rewrote README with current status analysis
- Updated .gitignore to protect Firebase credentials
- Enhanced footer component to support external links

### Removed
- Removed demo HTML files from root directory
- Removed scattered documentation files from root
- Cleaned up root directory to essential files only

## [0.1.0] - 2025-11-18

### Added
- Multi-AI comparison (GPT-3.5, Gemini, DeepSeek)
- 6-step analysis pipeline (preprocessing, bias, sentiment, ideology, topics, transparency)
- Hybrid architecture (JavaScript + optional Python ML)
- Consensus live debate system
- Export to YAML/JSON/PDF/README
- Grok-inspired dark theme UI
- Collapsible sidebar with conversation history
- AI model selector
- Pipeline analysis visualization
- Model synthesis view
- Timeline navigator
- Bias, tone, and fact check indicators
- Python ML service with spaCy, Detoxify, BERTopic
- Change detection system
- Transparency ledger foundation

### Technical Achievements
- React 18 frontend with Vite
- Node.js Express backend
- Flask Python ML service
- Multi-model AI integration
- Provenance tracking
- Audit trail logging
- Complete documentation
